## 英语阅读错题本

纯前端英语学习工具，帮助通过阅读积累词汇与语法知识。

### 功能

- **AI 分析** — 粘贴英文段落，自动生成翻译、词汇分解表和逐句语法分析
- **间隔复习** — 基于艾宾浩斯遗忘曲线的 12 级复习计划
- **浏览与管理** — 全文搜索、标签筛选、编辑和删除
- **学习统计** — 掌握进度、标签分布、近 30 天复习趋势

### 使用方式

直接用浏览器打开 `english-reader.html` 即可。

1. 在 **设置** 页配置 AI 提供商和 API Key
2. 在 **添加错题** 页粘贴英文原文，点击 AI 分析
3. 在 **复习** 页按间隔计划复习

### 支持的 AI 提供商

DeepSeek / 百度千帆 / 通义千问 / 智谱 GLM / Moonshot (Kimi) / OpenAI / 自定义端点

### 数据存储

所有错题数据存储在浏览器 localStorage 中。可随时通过设置页导出/导入 JSON 备份。

### 隐私安全

- API Key 支持 **仅本次会话** 模式（sessionStorage），关闭页面自动清除
- 所有数据仅存于本地浏览器，不上传任何服务器
